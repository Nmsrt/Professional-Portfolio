# Neo Portfolio - Wide Layout Revision

This is a revised React + Vite portfolio build with a wider, more consistent layout across the whole page.

## What changed

- widened the main layout container so sections use more of the screen
- removed the cramped centered feel in Experience, Tech, Projects, and Contact
- made section headers and content align more consistently
- improved large-screen spacing while keeping the site responsive
- included placeholder assets so the project runs immediately

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Replace `public/assets/AntonioMonserrat_CV.pdf` with your actual CV.
- Replace `public/assets/profile-illustration.png` if you want your own portrait image.
- Replace any placeholder icon assets in `public/assets/icons/` with your preferred logos.
