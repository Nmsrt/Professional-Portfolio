# Neo Portfolio – React + Vite

A modern, responsive personal portfolio website built with React and Vite. This project is designed to showcase projects, technical skills, and professional experience using a clean and wide layout.

---

## Overview

This is a one-page portfolio website focused on:

- Presenting academic and personal projects
- Highlighting technical skills and tools
- Showcasing experience and career journey
- Providing clear contact access for opportunities

The layout emphasizes full-width usage to improve readability and visual balance across different screen sizes.

---

## Key Features

- Wide layout design for better use of screen space
- Consistent section structure and alignment
- Responsive design across devices
- Component-based React architecture
- Clean dark-themed interface with subtle accents
- Fast development workflow powered by Vite

---

## Tech Stack

- Frontend: React 18
- Build Tool: Vite
- Styling: Custom CSS
- Assets: Local /public directory

---

## Project Structure

├── public/

├── src/

│   ├── main.jsx       # React entry point

└── README.md

---

## Installation

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Build

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

---

## Assets and Customization

Before deploying, update the following:

- Replace public/assets/AntonioMonserrat_CV.pdf with your actual resume
- Replace public/profile.png with your personal image
- Update icons in public/assets/icons/
